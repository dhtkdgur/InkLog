# 🖋️ Inklog (잉크로그) — 나만의 기록 중심 감상문 서비스

> **"당신의 감상을, 잉크처럼 남기세요."**  
> Inklog는 영화, 도서, 웹툰 등 다양한 문화 콘텐츠에 대한 개인적인 감상과 기록을 간편하고 아름답게 보관할 수 있는 **로컬 기반 감상문 아카이빙 서비스**입니다.

---

## 📌 1. 프로젝트 개요

일반적인 평점 사이트(왓챠피디아, Goodreads 등)가 '타인과의 평점 공유와 커뮤니티'에 방점을 둔다면, **Inklog**는 **'나만의 주관적인 기록과 아카이빙'**에 중점을 둔 개인용 공간입니다. 
서버 없이 브라우저의 로컬 저장소(`localStorage`)만을 활용하여 동작하므로, 별도의 회원가입 없이 즉시 나만의 기록을 쌓아나갈 수 있습니다.

- **목적**: 개인 문화생활(영화, 책, 웹툰, 기타) 아카이빙 및 시각적 통계 제공
- **개발 환경**: 순수 HTML5, CSS3, JavaScript (Vanilla JS) 및 HTML5 UP 템플릿 기반
- **주요 특징**: 로컬 저장소 활용, 완전한 클라이언트 사이드 동작, 데이터 백업/복원(JSON 내보내기/가져오기) 기능 지원

---

## 🛠️ 2. Technical Stack & Libraries

- **Frontend Core**: HTML5, CSS3, JavaScript (ES6+)
- **UI & Layout**: HTML5 UP - Editorial 템플릿 기반 커스텀 스타일링 (`custom.css`)
- **Icon Pack**: FontAwesome v6 (Solid, Regular)
- **Data Layer**: Web Storage API (`localStorage`)
- **Helper Libraries**: jQuery, browser.js, breakpoints.js, util.js (템플릿 레이아웃 반응형 보조용)

## ✨ 3. 주요 화면 및 기능 안내

Inklog는 총 **8개의 화면**으로 구성되어 있으며, 각각 유기적으로 데이터를 주고받습니다.

### 🏠 홈 대시보드 (`index.html`)
- **대시보드 통계**: 전체 및 카테고리별(영화, 책, 웹툰, 기타) 감상문 누적 개수를 부드러운 카운트업 애니메이션으로 표기합니다.
- **최신 감상문**: 최근에 작성한 감상문 3개를 카드 레이아웃으로 빠르게 확인합니다.
- **통계 그래프**: 내가 준 별점의 분포(막대 차트)와 최근 6개월간 월별 작성 추이를 동적 그래프로 렌더링합니다.
- **태그 클라우드**: 사용 빈도가 높은 태그 15개를 빈도수에 따라 폰트 크기가 다르게 시각화합니다.
- **최고의 기록 (Best Picks)**: 내가 남긴 기록 중 가장 평점이 높은 글(동률 시 최신순)을 추천합니다.
- **오늘의 한 줄 (Daily Quote)**: 등록된 감상문 중 '도서 인용구'가 포함된 글에서 무작위로 구절을 뽑아 대문짝만하게 보여줍니다.

### 📋 전체 기록 목록 (`list.html`)
- **카테고리 필터링**: 전체, 영화, 책, 웹툰, 기타 카테고리를 탭 형태로 분류하여 조회합니다.
- **실시간 검색**: 제목, 크리에이터(감독/작가), 한줄평, 본문, 태그를 아우르는 통합 실시간 키워드 검색을 지원합니다.
- **정렬 시스템**: 작성일(최신순), 평점순(높은순/낮은순)으로 목록을 즉시 정렬할 수 있습니다.

### ✍️ 감상문 작성 & 수정 (`write.html` / `edit.html`)
- **반응형 폼 유효성 검사**: 필수 입력 항목(카테고리, 제목, 한줄평, 본문, 별점) 누락 시 즉각적인 에러 메시지를 제공합니다.
- **동적 필드 노출**: 카테고리에서 **'책'**을 선택하면 인상 깊었던 구절을 적을 수 있는 '인용구/명대사' 입력란이 활성화됩니다.
- **별점 인터랙션**: 마우스 호버와 클릭으로 직관적인 5단계 별점 평가를 입력합니다.
- **태그 관리**: 쉼표로 태그를 입력하면 하단에 실시간으로 태그 배지가 렌더링되며, 기존에 등록한 태그들을 기준으로 브라우저 자동완성(`datalist`)을 추천합니다.

### 🔍 상세 보기 (`detail.html`)
- **카드 및 비주얼 요소**: 커버 이미지(미제공 시 카테고리별 기본 일러스트로 자동 대체), 평점, 제목, 한줄평, 본문이 깔끔한 잡지 형태로 렌더링됩니다.
- **추가 메타 정보**: 관람 날짜, 관람 장소, 재감상 여부(아이콘 표시), 본문 글자 수(공백 제외)가 계산되어 표시됩니다.
- **관련 기록 추천 (스마트 알고리즘)**: 현재 글의 태그들과 겹치는 태그 개수를 계산하고, 동일 카테고리 가중치를 주어 관련도가 높은 추천 감상문 3개를 하단에 노출합니다.
- **이전/다음 글 탐색**: 목록으로 돌아가지 않고도 이전 작성글과 다음 작성글로 이동할 수 있는 내비게이션 링크를 제공합니다.
- **공유 및 관리**: 현재 상세 페이지의 URL을 복사하는 링크 복사 기능과 수정/삭제 기능을 지원합니다.

### 🏷️ 태그 허브 (`tag.html`)
- **전체 태그 시각화**: 작성된 모든 감상문에서 추출한 태그들을 빈도수에 맞춰 글자 크기와 불투명도로 시각화한 대형 태그 클라우드입니다.
- **태그 랭킹**: 가장 많이 언급된 인기 태그 Top 5 순위표를 보여줍니다.
- **연동 목록**: 특정 태그를 선택하면 해당 태그가 달린 감상문 목록과 해당 태그가 카테고리별로 몇 개씩 속해 있는지 통계를 실시간으로 출력합니다.

### 🖼️ 비주얼 갤러리 (`gallery.html`)
- 감상문의 커버 이미지만 모아 바둑판 배열로 보여주는 반응형 갤러리입니다.
- 각 이미지 호버 시 작품 제목과 평점(별표 문자)이 오버레이 레이어로 감각적이게 나타납니다.

### 💾 소개 및 데이터 관리 (`about.html`)
- **데이터 백업 (Export)**: 로컬 저장소에 저장된 감상문 전체 데이터를 날짜 스탬프가 찍힌 `inklog-backup-YYYY-MM-DD.json` 파일로 다운로드합니다.
- **데이터 복원 및 병합 (Import)**: 백업된 JSON 파일을 불러와 기존 데이터와 합칩니다. 중복된 고유 ID(`id`)를 가진 데이터는 새로운 데이터로 덮어쓰고, 새로운 데이터는 목록 처음에 배치하여 시간순으로 정렬 후 로컬 저장소에 반영합니다.

## 💡 4. 프로젝트 개선점 제안 (Refactoring & Feature Proposals)

Inklog는 로컬 환경에 완벽히 적합한 훌륭한 기능들을 가지고 있으나, 향후 고도화를 위해 아래와 같은 개선점을 적용할 수 있습니다.

### 1️⃣ 아키텍처 및 코드 구조 개선 (모듈화)
* **현황**: 현재 모든 페이지의 초기화와 이벤트 바인딩 로직이 단 하나의 대형 스크립트 파일 `app.js` (약 930줄)에 통합되어 있습니다. `DOMContentLoaded` 시점에 단순 URL 판별 방식으로 라우팅 처리를 하고 있습니다.
* **개선안**: ES6 모듈(`import`/`export`)을 활용하여 각 페이지 전용 모듈(`index.js`, `detail.js`, `write.js`, `tag.js` 등)로 분리하고, 공통 헬퍼(예: `esc`, `formatDate`, `starsHtml`) 역시 `utils.js`로 분할 관리합니다. 코드 가독성과 유지보수성이 대폭 향상됩니다.

### 2️⃣ 데이터 저장소 고도화 (IndexedDB / Cloud Sync)
* **현황**: `localStorage`는 브라우저 캐시 삭제 시 데이터가 소실되며, 저장 한계 용량이 약 5MB로 제한되어 있습니다.
* **개선안**:
  - 로컬 환경 유지를 고수할 경우: 저장 용량이 유연하고 비동기로 작동하는 **`IndexedDB`**를 활용하여 데이터 저장 레이어를 리팩토링합니다.
  - 다중 기기 동기화 희망 시: Firebase, Supabase, 혹은 WebDAV API를 연동하여 기기 간 실시간 연동 및 동기화를 지원합니다.

### 3️⃣ 사용자 경험(UX) 고도화 기능
* **현황**: 커버 이미지는 외부 URL 주소를 직접 복사해 넣어야 하는 번거로움이 있습니다.
* **개선안**: 
  - **드래그 앤 드롭 및 로컬 이미지 임베드**: HTML5 File API와 `FileReader`를 이용해 컴퓨터 안의 이미지를 등록할 수 있게 하거나, Cloud 스토리지 업로드 연동을 구성합니다.
  - **인터랙티브 태그 입력기 (Chip UI)**: 현재 단순 쉼표 문자열 입력 방식 대신, 엔터 혹은 쉼표 입력 시 즉시 개별 태그 칩(Chip) 오브젝트로 변환되는 UI(예: 태그 삭제 `x` 버튼 포함)를 도입하여 시각적 직관성을 향상합니다.
  - **에디터 고도화**: 단순 본문 textarea 대신, Markdown 혹은 WYSIWYG 에디터를 가볍게 내장하여 볼드, 이탤릭, 링크, 구분선 등을 감상문 본문에 직접 스타일링할 수 있게 개선합니다.
  - **다크 모드 (Dark Mode)**: 야간 독서 및 영화 감상 후 눈 피로를 덜어주기 위한 감성적인 다크 테마 시스템(Theme Switcher)을 제공합니다.
