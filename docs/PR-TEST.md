# 🚀 풀 리퀘스트(PR) 연습용 테스트 파일

본 파일은 깃허브에서 직접 **풀 리퀘스트(Pull Request)**를 생성하고 연습해보기 위해 작성된 테스트용 문서입니다.

## 📝 PR을 여는 방법

1. 깃허브 저장소([dhtkdgur/InkLog](https://github.com/dhtkdgur/InkLog))로 이동합니다.
2. 상단에 나타나는 **"Compare & pull request"** 초록색 버튼을 누릅니다.
3. 만약 버튼이 보이지 않는다면, **Pull Requests** 탭으로 이동한 뒤 **New Pull Request**를 누릅니다.
4. `compare: feature/pr-practice` 브랜치를 선택한 뒤 타이틀과 설명을 적고 **Create pull request**를 완료합니다!
